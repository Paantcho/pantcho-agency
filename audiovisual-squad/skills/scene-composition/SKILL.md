# Scene Composition

## Missão
Transformar uma ideia em briefing visual técnico que o Engenheiro de Prompts traduz para linguagem de modelo.

## Elementos da Composição

### Câmera
- **Ângulo:** eye level / low angle / high angle / dutch angle
- **Distância:** close-up / medium shot / wide shot / extreme close-up
- **Lente:** 35mm (amplo, ambiental) / 50mm (natural) / 85mm (retrato, bokeh)
- **Foco:** sujeito em foco / fundo desfocado (bokeh) / tudo em foco

### Iluminação
- **Natural:** golden hour / blue hour / overcast / harsh midday
- **Direção:** frontal / lateral / contraluz / rembrandt
- **Qualidade:** suave e difusa / dura e direta / dramática

### Composição Visual
- **Regra dos terços:** posicionar sujeito em pontos de interesse
- **Linhas guia:** usar elementos do ambiente para conduzir o olhar
- **Profundidade:** foreground / midground / background
- **Espaço negativo:** respiro ao redor do sujeito

## Template de Briefing de Cena
```
AMBIENTE: [interno/externo, local específico da bible]
HORA DO DIA: [período, tipo de luz]
CÂMERA: [ângulo, distância, lente estimada]
SUJEITO: [posição, ação, expressão]
ROUPA: [descrição completa]
PROPS: [o que ela segura/usa]
FUNDO: [o que aparece, desfocado ou não]
MOOD: [adjetivos — intimista, luminoso, melancólico...]
```
