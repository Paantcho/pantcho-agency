# Consistency Validation

## Missão
Nenhum prompt passa sem validação. Cada detalhe verificado contra a bible.

## Checklist de Validação

### Aparência Física
- [ ] Tom de pele correto
- [ ] Cor e textura do cabelo corretos
- [ ] Comprimento do cabelo correto
- [ ] Cor dos olhos correta
- [ ] Tatuagens documentadas presentes/ausentes corretamente
- [ ] Proporções corporais alinhadas com referências

### Ambiente
- [ ] Ambiente está na lista de aprovados?
- [ ] Mobiliário travado correto?
- [ ] Vegetação/clima compatível com localização?
- [ ] Nenhum elemento anacrônico?

### Estética
- [ ] Iluminação coerente com o mood definido?
- [ ] Paleta de cores alinhada com identidade?
- [ ] Estilo de roupa dentro do guarda-roupa documentado?

## Resultado
- ✅ APROVADO → passar ao Engenheiro de Prompts
- ❌ REPROVADO → listar cada violação com correção sugerida
