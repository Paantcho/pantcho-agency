# Creator Bible

## Missão
Garantir que todo agente trabalhe com a mesma referência de identidade. A bible é a fonte da verdade — nenhum detalhe é inventado ou assumido.

## Como Usar
1. Identificar qual creator está ativa (verificar WORKING.md)
2. Carregar `audiovisual-squad/memory/creators/[nome]/APPEARANCE.md`
3. Carregar `audiovisual-squad/memory/creators/[nome]/AMBIENTES.md`
4. Só então executar a tarefa

## Regra de Ouro
Se um detalhe não está documentado → PARAR e perguntar. Nunca inventar características de aparência ou ambiente.
