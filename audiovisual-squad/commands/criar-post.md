# /criar-post
Trigger: criar post, fazer post, conteúdo para hoje, post para o feed

## Fluxo
Orquestrador → identifica creator ativa

Audiovisual Squad:
1. Planner → ideia e contexto narrativo
2. Copywriter → legenda e copy
3. Diretor de Arte → mood e estética
4. Diretor de Cena → composição técnica da cena
5. Consistência → validação (veto se necessário)
6. Eng. Prompts → prompt final de imagem + vídeo

Output: copy pronta + prompt pronto para geração
