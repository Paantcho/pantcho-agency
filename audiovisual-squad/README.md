# Audiovisual Squad

Criação de conteúdo visual (foto e vídeo) para creators digitais com consistência absoluta.

## Agentes

| Agente | Função |
|--------|--------|
| Planner | Calendário, narrativa, ideias |
| Copywriter | Voz da creator — legendas, scripts |
| Diretor de Arte | Mood, paleta, estética |
| Diretor de Cena | Composição, luz, ambiente, props |
| Especialista em Consistência | Guardião da identidade — poder de veto |
| Engenheiro de Prompts | Prompt final para geração |

## Dois Modos de Operação

**Modo ideia → post:**
Planner → Copywriter → Diretor de Arte → Diretor de Cena → Consistência → Eng. Prompts

**Modo referência → post com creator:**
Diretor de Cena (reference-deconstruction) → Diretor de Arte → Copywriter → Consistência → Eng. Prompts
